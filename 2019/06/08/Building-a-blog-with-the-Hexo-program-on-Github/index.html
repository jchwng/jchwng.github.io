安装前的准备

部署 Hexo 需要在电脑中预先安装好两个软件：Git和 Node.js。它们使电脑具备了与 Github 网站上的代码取得交流的环境，是一种 版本控制 工具，让网站上的代码与个人电脑上的保持同步，并通过命令行的方式创建与更新博客。关于它们的安装与使用，可以参考官方网站上的信息。
